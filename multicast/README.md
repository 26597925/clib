multicast
----------

a UDP multicast demo in Linux

Building
----------

    $ gcc -o multicast.out multicast.c    

Usage
----------

    $ ./multicast.out [-c as client] [-s as server] [-h help] 
    example:
        $ ./multicast.out -s

    Pls open 2 shell command window

    $ ./multicast.out -s
    $ ./multicast.out -s
